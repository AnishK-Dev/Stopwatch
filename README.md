# Stopwatch
A sleek, responsive web stopwatch with millisecond precision. Built with [HTML/CSS/JS], featuring high-performance logic for start, stop, and reset functionality. Designed for a clean user experience and mobile-first accessibility.
This stopwatch have a Start Button to start timer and a Stop Button to pause it and finally a Reset Button to reset the timer to 00:00:00:00.
This stopwatch have a friendly UI/UX design.
